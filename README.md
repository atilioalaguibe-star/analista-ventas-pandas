# 📊 Análisis de Ventas de Productos con Python

## 📌 Descripción del proyecto

Este proyecto consiste en un análisis exploratorio de datos de ventas de productos tecnológicos utilizando Python y la librería Pandas.

El objetivo es practicar el proceso completo de análisis de datos: desde la creación del dataset hasta la obtención de conclusiones basadas en métricas y visualizaciones.

---

## 🎯 Objetivos

- Crear y manipular un dataset de ventas
- Calcular métricas clave de negocio
- Analizar productos más rentables
- Visualizar resultados con gráficos
- Generar conclusiones basadas en datos

---

## 🛠️ Tecnologías utilizadas

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📊 Análisis realizado

Durante el proyecto se realizaron las siguientes tareas:

- Creación de un dataset de ventas
- Cálculo de ingresos por producto
- Agrupación de datos por producto
- Ordenamiento de productos por rentabilidad
- Visualización de ingresos mediante gráficos de barras

---

## 📈 Principales resultados

- Se identificó el producto más rentable del dataset
- Se observó la diferencia de ingresos entre productos
- Se calcularon métricas como total de ingresos y promedio de ventas

---

## 📌 Conclusión

Este análisis permite comprender cómo se comportan las ventas de distintos productos y cómo los datos pueden utilizarse para tomar decisiones informadas en un negocio.
